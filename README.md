# 🐾 Sistema de Adoção de Animais com Dashboard

Projeto acadêmico desenvolvido para facilitar e organizar o cadastro e a adoção de animais resgatados, conectando focinhos disponíveis aos seus futuros tutores e oferecendo um painel de indicadores em tempo real.

## 🎯 O Projeto
Este sistema foi criado com foco na usabilidade e agilidade, permitindo que ONGs ou protetores independentes gerenciem seus animais de forma simples e direta, além de monitorar as estatísticas gerais de cadastros.

## ✨ Funcionalidades
* **📊 Painel de Estatísticas (Dashboard):** Indicadores automatizados na barra lateral mostrando o total de animais, número de cachorros e número de gatos cadastrados em tempo real.
* **📝 Cadastro Completo:** Registro de animais com nome, espécie, porte, descrição comportamental, contato do responsável e upload de fotografia.
* **🐶 Galeria de Adoção:** Visualização dos animais disponíveis através de cards intuitivos, limpos e organizados.
* **🔍 Filtros Dinâmicos:** Pesquisa rápida de animais por Espécie e Porte, facilitando o "match" perfeito para o adotante.
* **💾 Banco de Dados Integrado:** Armazenamento seguro e persistente das informações utilizando SQLite integrado diretamente ao Python.

## 🛠️ Tecnologias Utilizadas
* **Python** (Linguagem principal)
* **Streamlit** (Framework para construção da interface web, métricas e design)
* **SQLite** (Banco de Dados relacional integrado)

## 🚀 Como executar o projeto localmente

1. Clone este repositório:
`git clone https://github.com/mariasoturna/sistema-adocao-animais.git`

2. Instale as dependências necessárias:
`pip install -r requirements.txt`

3. Execute a aplicação:
`streamlit run app.py`
